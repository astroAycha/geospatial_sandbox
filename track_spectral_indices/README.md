Monitoring Spectral Indices:
==================================
This directory contains scripts and resources for monitoring spectral indices derived from satellite imagery. Spectral indices are useful for assessing various environmental parameters such as vegetation health, water quality, and soil properties.

The main components of this directory include:
- `track_spectral_indices.py`: A script to compute various spectral indices (e.g., NDVI, EVI, SAVI) from satellite imagery.
